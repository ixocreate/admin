Provides API endpoints for the [Kiwi Admin frontend application](https://github.com/kiwi-suite/admin-frontend).

## Installation

Install the package via composer:

```sh
composer require ixocreate/admin
```

## Testing

```sh
composer install --dev
phpunit
```
